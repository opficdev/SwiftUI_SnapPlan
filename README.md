## 최소 iOS 버전
- iOS 16

### 특징
- Firebase를 통한 사용자 데이터 저장
- FirebaseAuth를 통한 사용자 인증
- Combine를 현재 날짜 및 시간을 분 단위로 최신화 (RxSwift로 대체 가능)
- 라이트 / 다크 모드 지원
